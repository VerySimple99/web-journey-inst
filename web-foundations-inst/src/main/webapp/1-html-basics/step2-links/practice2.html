경로의 종류

절대경로 (Absolute Path)

완전한 주소를 모두 적는 방법:

네이버 절대경로

구글 검색 절대경로

상대경로 (Relative Path)

현재 파일 기준으로 상대적인 위치를 나타내는 방법:

같은 폴더의 page2.html로 이동

pages 폴더 안의 about.html로 이동

상위 폴더의 index.html로 이동

폴더 구조 예시

    website/
    ├── index.html (현재 파일)
    ├── page2.html
    ├── pages/
    │   ├── about.html
    │   └── contact.html
    └── parent.html
