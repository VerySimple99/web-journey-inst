1. 제목 태그의 계층구조

제목 태그는 h1부터 h6까지 총 6단계가 있습니다.

제목 태그 예시

h4 - 네 번째 수준 제목

h5 - 다섯 번째 수준 제목
h6 - 가장 작은 제목

제목 태그는 단순히 글자 크기가 아니라 문서의 의미적 구조를 나타냅니다.

4. 핵심 정리

HTML 문서의 기본 구조

<!DOCTYPE html> → <html> → <head> + <body>

기억해야 할 중요 태그들

제목: h1, h2, h3 (의미적 계층 구조 중요)

문단: p (내용을 의미있는 단위로 구분)

강조: strong (중요함), em (강조)

특수: code (코드), cite (작품명)

HTML 작성 원칙
